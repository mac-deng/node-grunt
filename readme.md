# 文件配置
## 首先参照[ grunt上手 ](http://mac-deng.github.com/2013/02/18/grunt/)安装grunt。
## 下载配置文件
	git clone https://github.com/mac-deng/node-grunt.git
## 安装依赖
	npm install 
## 用法

#### 压缩图片
	grunt jupianyipic
#### 压缩JS和CSS，会在目录生成site.js和site.css文件
	grunt jupianyi
#### 监听文件夹变化触发jupianyi命令
	grunt watch
